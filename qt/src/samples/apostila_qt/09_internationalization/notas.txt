1) É preciso criar dicionários para idiomas alvo

2) para isso todas as strings que devem
	ser traduzidas devem ser passadas
	para a função tr:
		  QMessageBox( this, tr("título"),
                        tr("texto"));

3) no arquivo de projeto inclua a seguinte linha:
TRANSLATIONS = app_pt.ts
onde "pt" é a sigla do idioma (ver tabela de idiomas)
podendo existir vários arquivos para vários  idiomas

4) execute o utilitário "lupdate nome_do_projeto " para gerar o arquivo ".ts"

5) utilize o QtLinguist, carregue o arquivo ".ts" para fazer as traduções
	e gerar o arquivo ".qm"
(OBS: no menu  File do QtLinguist já existe a opção "release"
 para gerar o arquivo ".qm", logo não e preciso utilizar o utilitário
"lrelease" na mão.
Então:
a) Traduza
b) salve
c) release

6)

a)coloque o arquivo ".qm" no arquivo de recursos

b) e faça o mesmo para o dicionário do qt
	para português já que não e garantido
	que ele estará no cliente.

7) no código, no main.cpp (veja exemplo)
instale os tradutores usando
o objeto QApplication.
